# 4. Sidebar

To toggle projects and tags sidebar press `control + cmd + s`

You can select and filter unlimited projects and tags.

## Each subfolder is a project

Make unlimited folders inside your storage: File -> New Folder or `option + cmd + n`.

Each project has its own settings, right click on the project -> Show Options.

![](assets/E0136C95-4331-4CC8-B510-20B638D5E6A1.png)

You can configure sorting, direction, visibility settings.

![](assets/B2A19BC7-7E42-4611-9808-51EDEE846BDF.png)

Or add an external folder: File -> Add External Fodler...
